

#  概述

通过Vue3和threejs打造三维场景 通过不同场景和商品的切换展示3D效果

## 开发

```
# 克隆项目
git clone https://gitee.com/chenpeng991105/vue_shop.git

# 进入项目目录
cd vue_shop

# 安装依赖
npm install

# 建议不要直接使用 cnpm 安装依赖，会有各种诡异的 bug。可以通过如下操作解决 npm 下载速度慢的问题
npm install --registry=https://registry.npm.taobao.org

# 启动服务
npm run serve
```
